## How to Run

### Prerequisites
- Python 3.x installed
- No external packages required

### Run Matrix Demo
- python matrix_demo.py

### Run Cocktail Sort Demo
- python cocktail_sort_demo.py

### Run Combined Demo
- python matrix_cocktail_sort.py

## Features Demonstrated

### Matrix
- Create matrix from 2D list
- Get/set individual elements
- Matrix addition and subtraction
- Matrix multiplication
- Scalar multiplication
- Matrix transpose

### Cocktail Sort
- Bidirectional bubble sort
- Forward and backward passes
- Time complexity: O(n^2) worst/average, O(n) best
- Step-by-step visualization

### Combined Application
- Flatten matrix to 1D list
- Sort using cocktail sort
- Rebuild sorted matrix
- Sort each row or column individually
